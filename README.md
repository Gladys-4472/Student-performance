# Student Performance Analysis
End-to-End Data Analysis Project Using Excel, SQL & Power BI
# Project Overview
<p>This project presents a comprehensive end-to-end analysis of student academic performance data.<br>
The objective was to identify key factors influencing student outcomes and transform raw data into actionable insights using structured data cleaning, statistical analysis, and interactive dashboard development.<br>
This project demonstrates practical skills in:<br>
Data cleaning and preparation<br>
Statistical analysis<br>
SQL querying<br>
Data modeling<br>
Business intelligence reporting</p>

# Project Objectives
The analysis aimed to answer the following questions:
  <p>Does study time significantly impact academic performance?<br>
  Is there a performance difference based on gender?<br>
  How does attendance rate influence final scores?<br>
  Which subjects have the highest and lowest average performance?<br>
  What patterns can be identified using correlation and regression analysis?</p>

# Tools & Technologies Used
- Microsoft Excel – Data cleaning, descriptive statistics, correlation & regression analysis [Download here]{https://www.microsoft.com}
- Microsoft SQL Server – Data querying, aggregation, segmentation analysis
- Microsoft Power BI – Data modeling, DAX measures, dashboard visualization
 - GitHub – Project documentation and version control

# Dataset Description
<p>The dataset contains structured student performance records including:<br>
Student ID<br>
Gender<br>
Subject<br>
Study Hours per Week<br>
Attendance Rate (%)<br>
Previous Grade<br>
Final Exam Score</p>
<p>The dataset was cleaned to remove inconsistencies, ensure correct data types, and validate numerical ranges before analysis.</p>

# Data Cleaning & Preparation
<p>Data preprocessing steps included:<br>
Removal of duplicates<br>
Standardization of categorical values<br>
Data type correction (numeric, percentage, text)<br>
Validation of missing values<br>
Creation of calculated columns where necessary<br>
In Power BI, a clean data model was built following best practices.</p>

# Statistical Analysis Performed
## Descriptive Statistics
<p>Mean<br>
Median<br>
Standard Deviation<br>
Minimum & Maximum values</p>
  
## Correlation Analysis
<p>Pearson correlation was used to evaluate relationships between:<br>
Study Hours & Final Score<br>
Attendance Rate & Final Score</p>

  ## Hypothesis Testing
<p>Tested whether:<br>
Study hours significantly impact performance<br>
Attendance significantly affects academic outcomes</p>

  ## Regression Analysis
<p>Linear regression was performed to measure predictive influence of:<br>
Study Hours<br>
Attendance Rate<br>
on Final Exam Score.</p>

  ## SQL Analysis
<p>SQL queries were used to:<br>
Calculate average score by gender<br>
Identify top-performing subjects<br>
Group students by performance level<br>
Aggregate performance metrics<br>
Perform segmentation analysis<br>
This demonstrated ability to manipulate structured data using T-SQL.</p>

  ## Power BI Dashboard Features
<p>The interactive dashboard includes:<br>
KPI Cards (Average Score, Total Students, Pass Rate)<br>
Performance by Gender (Clustered Column Chart)<br>
Subject Performance Comparison (Bar Chart)<br>
Study Hours vs Score (Scatter Plot)<br>
Dynamic Slicers (Gender, Subject)<br>
The dashboard allows interactive filtering for deeper insights.</p>

## Key Insights
<p>Study hours show a positive relationship with academic performance.<br>
Higher attendance rates are associated with better final exam outcomes.<br>
Some subjects consistently demonstrate lower average scores.<br>
Performance patterns vary slightly across gender groups.</p>

## Recommendations
<p>Encourage structured weekly study schedules.<br>
Improve attendance monitoring to reduce absentee impact.<br>
Provide academic intervention in lower-performing subjects.</p>
