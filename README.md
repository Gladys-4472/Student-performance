# Student-performance
The project analyzes student academic performance data to identify factors influencing scores and drive actionable insights for academic improvement
# Student Performance Analysis
End-to-End Data Analysis Project Using Excel, SQL & Power BI
# Project Overview
This project presents a comprehensive end-to-end analysis of student academic performance data.
The objective was to identify key factors influencing student outcomes and transform raw data into actionable insights using structured data cleaning, statistical analysis, and interactive dashboard development.
This project demonstrates practical skills in:
Data cleaning and preparation
Statistical analysis
SQL querying
Data modeling
Business intelligence reporting

# Project Objectives
The analysis aimed to answer the following questions:
  <p>Does study time significantly impact academic performance?<br>
  Is there a performance difference based on gender?<br>
  How does attendance rate influence final scores?<br>
  Which subjects have the highest and lowest average performance?<br>
  What patterns can be identified using correlation and regression analysis?</p>

# Tools & Technologies Used
<p>Microsoft Excel – Data cleaning, descriptive statistics, correlation & regression analysis<br> Microsoft SQL Server – Data querying, aggregation, segmentation analysis<br> Microsoft Power BI – Data modeling, DAX measures, dashboard visualization<br> GitHub – Project documentation and version control</p>

# Dataset Description
The dataset contains structured student performance records including:
Student ID
Gender
Subject
Study Hours per Week
Attendance Rate (%)
Previous Grade
Final Exam Score
<p>The dataset was cleaned to remove inconsistencies, ensure correct data types, and validate numerical ranges before analysis.</p>

# Data Cleaning & Preparation
Data preprocessing steps included:
Removal of duplicates
Standardization of categorical values
Data type correction (numeric, percentage, text)
Validation of missing values
Creation of calculated columns where necessary
In Power BI, a clean data model was built following best practices.

# Statistical Analysis Performed
## Descriptive Statistics
Mean
Median
Standard Deviation
Minimum & Maximum values
## Correlation Analysis
Pearson correlation was used to evaluate relationships between:
Study Hours & Final Score
Attendance Rate & Final Score
## Hypothesis Testing
Tested whether:
Study hours significantly impact performance
Attendance significantly affects academic outcomes
## Regression Analysis
Linear regression was performed to measure predictive influence of:
Study Hours
Attendance Rate
on Final Exam Score.
## SQL Analysis
SQL queries were used to:
Calculate average score by gender
Identify top-performing subjects
Group students by performance level
Aggregate performance metrics
Perform segmentation analysis
This demonstrated ability to manipulate structured data using T-SQL.
## Power BI Dashboard Features
The interactive dashboard includes:
KPI Cards (Average Score, Total Students, Pass Rate)
Performance by Gender (Clustered Column Chart)
Subject Performance Comparison (Bar Chart)
Study Hours vs Score (Scatter Plot)
Dynamic Slicers (Gender, Subject)
The dashboard allows interactive filtering for deeper insights.

## Key Insights
Study hours show a positive relationship with academic performance.
Higher attendance rates are associated with better final exam outcomes.
Some subjects consistently demonstrate lower average scores.
Performance patterns vary slightly across gender groups.

## Recommendations
Encourage structured weekly study schedules.
Improve attendance monitoring to reduce absentee impact.
Provide academic intervention in lower-performing subjects.
